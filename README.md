# 🏠 House Price Prediction

A data science project to predict house prices based on features like location, size, quality and condition.

## 📌 What I Did
Analyzed a real house price dataset and went through the complete data science pipeline from raw data to clean, model-ready data.

## 📂 Project Notebooks
- 📊 **Exploratory Data Analysis** - Explored the dataset, found patterns and visualized correlations using heatmaps, bar charts and distribution plots
- ⚙️ **Feature Engineering** - Discovered and handled missing values, removed outliers and transformed skewed features
- 🎯 **Feature Selection** - Reduced features from 80 to 20 most important ones that impact house prices
